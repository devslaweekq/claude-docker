# Changelog

This folder collects changelog entries, one Markdown file per change, named
`YYYY-MM-DD-<short-topic>.md`. Keeping each change in its own file avoids merge
conflicts on a shared `CHANGELOG.md` and keeps history easy to scan.

Entries follow the [Keep a Changelog](https://keepachangelog.com/) categories:
`Added`, `Changed`, `Deprecated`, `Removed`, `Fixed`, `Security`.
